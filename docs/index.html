Cinser

An operating system for people who value control, freedom, and understanding over abstraction.

View on GitHub Download Now Documentation

Hardware Support

Focused on the x86 family — starting with i386, with a path to x86_64. Boots via GRUB + Multiboot and aims to stay simple, transparent, and hackable.

Custom Core

Fully original and proprietary system. A highly optimized monolithic kernel, written from scratch in C and ASSEMBLY for maximum raw performance.

Open Source

Open-source under the GNU GPLv3 — contribute, fork, and learn freely.
