Installation

You can run Cinser directly on hardware or via an emulator like QEMU. The build system uses GRUB + Multiboot, freestanding C, and NASM.

Quick Start with QEMU

bash — cinser-dev 
# 1) Clone the repository
git clone https://github.com/voidunix/Cinser.git
cd Cinser

# 2) Install build dependencies (Debian/Ubuntu example)
sudo apt update
sudo apt install build-essential nasm grub-pc-bin xorriso qemu-system-x86 mtools

# 3) Build & run
make all
make run

Dependencies Required

Ensure you have build-essential, nasm, grub-pc-bin, xorriso, qemu-system-x86, and mtools installed before building.
